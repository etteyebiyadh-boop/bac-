## 🎯 Pourquoi Bac Excellence?

Chaque année, **250,000+ élèves tunisiens** passent le BAC. Seulement **30%** obtiennent une mention. Bac Excellence change la donne.

### 🏆 Ce qui nous rend uniques:

- ✅ **Corrections IA Réalistes** - Notation selon les standards du Ministère (moyenne nationale: 11-12/20)
- ✅ **8 Modules Officiels** - Tout le programme 2026: Holidaying → Literary Texts
- ✅ **6 Langues Complètes** - Anglais, Français, Arabe, Espagnol, Allemand, Italien
- ✅ **Section-Specific** - Contenu adapté à ta section (Math, Sciences, Lettres...)
- ✅ **Examens Passés** - Archive nationale avec méthodologie et corrigés modèles

## ✨ Fonctionnalités

### 📝 Writing Lab AI
Soumets ta rédaction et reçois une correction instantanée:
- **Grammar Score** /20
- **Vocabulary Score** /20  
- **Structure Score** /20
- **Feedback détaillé** avec explications
- **Version corrigée** avec améliorations
- **Leçon recommandée** selon tes faiblesses

### 📖 Reading Comprehension
Maîtrise les 12 points de la section I:
- 16 passages de lecture (2 par module)
- Questions types BAC avec justification
- Vocabulaire contextuel expliqué
- Timed practice comme l'examen

### 🎓 BAC Calculator
Simule ta moyenne et optimise tes efforts:
- Toutes les sections: Math, Sciences, Lettres, Éco, Tech, Info
- Coefficients officiels du Ministère
- Target score calculator (Assez Bien → Très Bien)

### 🏅 Daily Missions
15 minutes par jour pour progresser:
- Missions générées selon tes points faibles
- Streak tracking pour motivation
- XP et gamification

### ⚔️ Challenges
Affronte tes amis:
- Défie quelqu'un sur un examen
- Comparaison de scores
- XP rewards (100 XP par victoire)

## 🚀 Commencer

### Prérequis
- Node.js 18+
- PostgreSQL database
- OpenAI API key

### Installation

```bash
# Cloner le repo
git clone https://github.com/etteyebiyadh-boop/bac-excellence.git
cd bac-excellence

# Installer les dépendances
npm install

# Configurer les variables d'environnement
cp .env.example .env
# Editer .env avec tes clés

# Générer Prisma client
npx prisma generate

# Pousser le schéma
npx prisma db push

# Seeder la base de données
npm run db:sync

# Lancer le serveur
npm run dev
```
